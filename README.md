Cat Clicker Game
Description:
A simple game where a cat catches falling items. The player can control the cat by clicking or using the arrow keys, manage its hunger, and collect coins.
Project Goal:
Learn the basics of JavaScript, HTML, and CSS
Practice animations, event handling, and DOM manipulation
Build interactive elements and game logic
Current Features:
Cat moves across the full width of the scene
Smooth movement towards click positions
Arrow key controls for the cat
Falling items (coins and food)
Coin counter and hunger level
“Restart” button to start the game over
Game over when hunger reaches 0
Shop to change cat skins
Future Improvements:
More complex levels and item types
Sounds and visual effects
Animated background (grass, clouds)
Additional skins and cat upgrades
Technologies Used:
HTML, CSS, JavaScript
Status: In progress — many features are yet to be added
Planned features: optimization of game mechanics, optimization of the shop, as well as more skins and additional features.
